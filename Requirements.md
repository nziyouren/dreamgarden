### 参赛核心要求


为确保比赛公平性与项目质量，所有参赛项目必须满足以下六项要求：



#### 📅 项目开始时间：

项目必须在2026年1月27日（含）之后开始创建。不允许提交既有项目的延展版本或历史代码的简单迁移与包装。

#### 项目赛道：

Stablelayer 赛道：Mac Mini *1

项目必须赋能Stablelayer或使用其SDK。

参考文档：

https://docs.stablelayer.site/

https://github.com/StableLayer/stable-layer-sdk


#### 🔍 Move语言版本：

智能合约必须使用 Move 最新语法，版本要求为 Move 2024。不符合规范的合约将不予评审。



#### ⚒️ 官方SDK要求：

项目必须使用 Sui 官方 SDK 的最新版本（如 TypeScript / Rust SDK），不得使用已弃用的官方接口。



#### ⚙️ 可运行产品与线上网站：

项目必须完整可运行，提供可访问的网站与能展示核心功能的真实 Demo。仅代码、仅合约或仅 PPT 的项目不被接受。



#### 🗳️ 开源要求：

项目代码必须完全开源，提供公共 GitHub/GitLab 仓库，至少包含智能合约代码、前后端核心逻辑及清晰的 README 部署说明。



#### 🤖 AI使用披露：

如项目使用任何 AI 工具（如 ChatGPT、Claude、Cursor 等），必须完整披露工具名称、模型版本及关键 Prompt 原文。
